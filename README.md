# Linux
```shell
pip3 install -r requirements.txt
# нужно зайти в файл bot/config/config.py и там указать API_ID и API_HASH
nano bot/config/config.py
python3 main.py
```

# Windows
```shell
pip install -r requirements.txt
# нужно зайти в файл bot/config/config.py и там указать API_ID и API_HASH
python main.py
```
